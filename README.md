## What This Is

Language isn't a domain you practice like basketball. Language is the MEDIUM OF MEANING. Every doc in every repo is written IN language. Language is the water the fish are swimming in.

When you say "black hole = gut" — that's a linguistic operation. Metaphor. Mapping. Translation. When you say "digestion IS comprehension" — that's language collapsing two concepts into one. This repo names how that works.
