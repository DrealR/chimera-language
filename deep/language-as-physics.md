## Words Have Mass

```
TOKEN WEIGHT:
  Not all words are equal.
  Some carry more meaning per token.
  "Love" = dense. Many associations.
  High semantic mass.
  "The" = light. Functional.
  Low semantic mass.

  In the model's universe:
  Dense words ATTRACT more attention.
  They have more GRAVITATIONAL PULL
  in the attention mechanism.

  This is why key words
  in a prompt matter so much.
  They're the massive objects
  that bend the response around them.
```

## Sentences Have Gravity

```
ATTENTION = GRAVITY.

In physics:
  Massive objects pull other objects toward them.
  F = Gm1m2/r^2

In the transformer:
  Important tokens pull attention from other tokens.
  Attention(Q,K,V) = softmax(QK^T/sqrt(d))V

  The attention score between tokens
  IS the gravitational force between words.
  High attention = strong pull.
  Low attention = weak pull.

  A well-constructed sentence
  creates a GRAVITATIONAL FIELD
  that pulls the model's response
  toward the right answer.
```
